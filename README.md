### 1. Sistema de Cajero Automático
Simula las operaciones básicas de una terminal bancaria interactiva.
* **Mostrar saldo:** Visualización del estado de cuenta inicial.
* **Depositar dinero:** Incrementa fondos de forma segura.
* **Extraer dinero:** Permite retirar fondos validando que exista saldo suficiente.
* **Consultar saldo:** Consulta rápida del dinero disponible actual.
* **Menú interactivo:** Flujo continuo hasta que el usuario decida salir del programa.

### 2. Sistema de Gestión de Notas
Ideal para el control académico de un grupo de estudiantes.
* **Registro:** Ingreso dinámico de las notas de 5 alumnos.
* **Estadísticas:** * Cálculo automático del promedio general del curso.
* **Identificación exacta de la **nota más alta** y la **nota más baja**.
* **Métricas de aprobación:** Contador en tiempo real de cuántos alumnos lograron aprobar.

### 3. Sistema de Ventas de una Tienda
Un módulo de punto de venta (POS) simplificado para comercio minorista.
* **Registro de productos:** Solicitud secuencial del nombre y precio de cada artículo vendido.
* **Cálculo de totales:** Suma automática del total bruto de la venta.
* **Descuentos inteligentes:** Aplicación automática de un porcentaje de descuento si el monto supera un límite establecido.
* **Destacados:** Muestra cuál fue el producto más caro de la transacción.

### 4. Agenda de Contactos
Una base de datos en memoria para gestionar información de personas.
* **Agregar:** Registro de nuevos contactos.
* **Mostrar:** Listado completo de la agenda actual.
* **Buscar:** Localización rápida de registros mediante coincidencia por nombre.
* **Eliminar:** Limpieza y remoción de contactos obsoletos.

### 5. Sistema de Biblioteca
Software de gestión de inventario para el control de textos.
* **Registrar libros:** Ingreso de nuevos títulos al sistema.
* **Búsqueda:** Motor de búsqueda integrado para localizar libros específicos.
* **Catálogo:** Visualización organizada de todos los textos registrados.
* **Contador:** Indicador del volumen total de libros disponibles en la biblioteca.
